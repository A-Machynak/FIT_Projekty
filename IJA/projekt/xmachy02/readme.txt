Pro přeložení projektu je požadován Apache Ant (viz build.xml):
 $ ant compile
- Přeložení - vytvoření .class souborů v ./dest/

 $ ant jar
- Vytvoří .jar soubor v ./dest/

 $ ant run
- Spustí aplikaci (a přeloží)

 $ ant all
- Spustí jar a doc

 $ ant clean
- Smazání složek: ./build/ ./dest/ a ./doc/

 $ ant doc
- Vytvoření dokumentace v ./doc/
